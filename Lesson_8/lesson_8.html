• 1. Пользователь вводит строку вида: 'firstName = John, email=example@gmail.com, balance=300; firstName=Test, lastName=Test, email=admin@gmail.com, balance=1000';
  • 2. Количество возможных полей не ограничено приведенными в примере
  • 3. Из полученной строки получить массив пользователей вида:
    var users = [{
    firstName: 'John',
    email: 'example@gmail.com',
    balance: 300
    }, {
    firstName: 'Test',
    lastName: 'Test',
    email: 'admin@gmail.com',
    balance: 1000
    }];

  • 3. Написать функцию groupBy(fieldName, usersList), которая группирует пользователей по указаному полю. Поле может указываться любое. пример: groupBy('firstName', users) должна вернуть результат вида:
    var groupedUsersExample = {
    'John': [{
